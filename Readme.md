# Calgary Air Quality Analysis ETL Project

## Project Overview
This project involves the extraction, transformation, and loading (ETL) of air quality data for the City of Calgary.

### Key Features
- ETL Pipeline
- Data Transformation
- Data Visualization

#### Technologies Used
- Apache Spark
- Python
- Jupyter Notebook

##### Project Workflow
1. Data Extraction
2. Data Transformation
3. Data Loading
4. Data Visualization
